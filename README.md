# AuditLensBuilder

1. `docker-compose up -d`

2. `ngrok http 5000`

3. open on mobile device



## Сторінка камери (CameraCapture)

1. Фіксований розмір відео: max-w-[414px], max-h-[736px] портретна орієнтація
2. Спрощений дизайн: видалено градієнти та анімації
3. Локація внизу: чорна смуга на всю ширину відео max-w-[414px] і відображає поточні отримані координати білим текстом. Верхнія лінія дотична без відступу до нихньої грані відео, і виглядає ніби його продовження.
4. Текстове поле: темне поле для введення коментаря на чорній смузі на всю ширину відео max-w-[414px]
5. По центру: біла кнопка Camera (знімок, неактивна без камери)
кнопка з правого боку темно сіра або синя кнопка Upload (вибір з галереї)
6. блок з кнопками внизу:
Ліворуч: відміна, повернення до попередньої сторінки




## Сторінка редагування (PhotoEditor)

1. Фіксований розмір фото отриманого з камери: max-w-[414px], max-h-[736px] портретна орієнтація
2. Спрощений дизайн: видалено градієнти та анімації
3. Локація внизу: чорна смуга на всю ширину відео max-w-[414px] і відображає поточні отримані координати білим текстом. Верхнія лінія дотична без відступу до нихньої грані відео, і виглядає ніби його продовження.
4. Текстове поле: темне поле для введення коментаря на чорній смузі на всю ширину відео max-w-[414px]
5. Зверху зліва: кнопка з хрестиком, по  суті відміна, повертає то сторінки Camera щоб зробити новий знімок без збереження
знизу зліва: кнопка з плюсом, для додавання на фото круглого стікера напівпрозорого в середині. Це єдиний стікер. клазнувши по ньому після додавання, можна змінити його розмір потягнувши за край рамки і перемістити натиснувши на нього і потягнувши
6. блок з кнопками внизу:
Ліворуч: відміна, повернення до попередньої сторінки
Праворуч пнопка публікації або збереження

після публікаціїх повертаємось до сторінки перекляду проекту

## Сторінка редагування (PhotoEditor)

1. Фіксований розмір фото: max-w-[414px] min-w-[414px], max-h-[736px] min-h-[736px]  - центроване на екрані     
Спрощений хедер: темний з фіолетовою іконкою
Фото по центру: на чорному фоні
Локація внизу: зелена смуга "Локація"
4. Назва проекту: чорна смуга з жовтою рамкою з назвою проекту жирним шрифтом (отримується з API) на всю ширину відео max-w-[414px]
Тексти внизу: жовта смуга "Тексти"
Стікери: 4 кнопки в ряд (Circle, Filled, Target, Arrow)
Трансформація: кнопки Rotate, Resize, Delete
Позиція коментаря: радіо-кнопки Top/Bottom
Додаток готовий до використання з новим дизайном!




## правки

1. текст локації має бути динамічний, той що визначено по gps.



2. **project name** - custom comment

project name - Назва поточного проекту до якого додається фото (отримується з API) 

custom comment - текст із Текстового поля (темне поле для введення коментаря), який ввів користувач

 



3. Timestamp: чорна смуга з датою/часом справа (формат: 0000-00-00 00:00:00) – тут записуємо дату час коли фото було зроблено а не толі коли було завантажено. Тобто коли робимо фото - запамґятовуєм час та дату і коли надсилатимем фото на рендер додамо туди такі дані: саме фото, локація що була визначена тоді коли було зроблене фото у форматі lat,lng , назва проекту у якому зараз додається фото, текстовий коментар що користувач ввів у текстовому полі та дату й час коли було зроблено фото і координати та розмір круглого червоного стікеру доданого на фото. Таким чином сервер просто відобразить ці дані
